What is Tails7?


Tails7 is an engine for mapping objects on the screen in the style of Mode7 (like from SNES) but it has additional features and is designed for Clickteam Fusion.


It supports basic functions like sprite support, of course, and less basic ones like dedicated modes for how an object should behave, Polygon Objects giving 3D walls and line objects. All calculations are shader-based.
