Tails7 Center Pointer


1. General Description of Object

Tails7 Center Pointer is a mapping helper object. It selects the closest "Tails7 Render Ground" object from the center when there may be several in the frame.

This object is a reference point that is always in the center of the screen.


2. Variables

⚠️ This object has no variables of its own.



3. Behaviors

The main behavior is Position, which handles positioning.

3.1. Details: Position in Tails7 Center Pointer


Position

* Always
Tails7 Center Pointer: Set X position to _RenderMiddleSizeX( "Tails7 Info Object" )
Tails7 Center Pointer: Set Y position to _RenderMiddleSizeY( "Tails7 Info Object" )
Tails7 Center Pointer: Move to layer _TargetZLayer( "Tails7 Info Object" )

Position X: Sets the X position in the middle of the screen.
Position Y: Sets the Y position in the middle of the screen.
Layer: Changes the position of the object on the layer where the object "Tails7 Render Ground" is located.


See also: How this pointer cooperates with render ground and mapping logic in Tails7 Mapper Object
