Sprite Objects


1. General Description of Object

Sprite Objects are, in other words, objects belonging to the Group.3D Objects. They enable on-screen mapping that gives depth.
All behaviors related to these objects are performed by the Tails7 Mapper Object.


2. Variables

Alterable Values
Description
_PositionX

Type: Numeric parameter.

Function: This variable specifies where the object should be positioned on the X-axis.

Default value: 0.0

_PositionY

Type: Numeric parameter.

Function: This variable specifies where the object should be positioned on the Y-axis.

Default value: 0.0

_OffsetX

Type: Numeric parameter.

Function: This variable shifts the object along the X-axis on the screen after the Mapping Mode calculations.

(Positive values shift to the right, negative values shift to the left.)

Default value: 0.0

_OffsetY

Type: Numeric parameter.

Function: This variable shifts the object along the Y-axis on the screen after the Mapping Mode calculations.

(Positive values shift down, negative values shift up.)

Default value: 0.0

_Scale

Type: Numeric parameter.

Function: This variable multiplies _ScaleX and _ScaleY values.

Default value: 1.0

_ScaleX

Type: Numeric parameter.

Function: This variable represents the object's scale along the X-axis.

Default value: 1.0

_ScaleY

Type: Numeric parameter.

Function: This variable represents the object's scale along the Y-axis.

Default value: 1.0

_LayerOffset

Type: Numeric parameter.

Function: This variable specifies the layer offset: negative values allow the object to render behind objects behind it, and positive values allow it to render above objects in front of it.

Default value: 0.0

_RenderDistance

Type: Numeric parameter.

Function: This variable determines when to set the _OutRenderDistance flag. If this value is less than _LayerDistance, it sets _OutRenderDistance to False; otherwise to True. Setting it to -1.0 disables changes to _OutRenderDistance (Note: this flag may indicate object visibility).

Default value: -1.0

_Distance

Type: Numeric parameter.

Function: This variable is automatically calculated by the Mapping Mode behavior; it computes the distance from _DistanceX and _DistanceY.

Default value: 0.0 (Automatic)

_DistanceX

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it computes the distance along the X-axis for the Mapping Mode calculations.

Default value: 0.0 (Automatic)

_DistanceY

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it computes the distance along the Y-axis for the Mapping Mode calculations.

Default value: 0.0 (Automatic)

_RotPosX

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it rotates the _PositionX point by the _RotZ value.

Default value: 0.0 (Automatic)

_RotPosY

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it rotates the _PositionY point by the _RotZ value.

Default value: 0.0 (Automatic)

_InY

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it computes the shader-like distortion of the point, based on the _Distortion variable.

Default value: 0.0 (Automatic)

_DistortedX

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it is based on _RotPosX but modified by _InY.

Default value: 0.0 (Automatic)

_DistortedY

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it is based on _RotPosY but modified by _InY.

Default value: 0.0 (Automatic)

_OutX

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it is based on _DistanceX but centered on the Tails7 Render Ground position.

Default value: 0.0 (Automatic)

_OutY

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it is based on _DistanceY but shifted to the Tails7 Render Ground position.

Default value: 0.0 (Automatic)

_FinalX

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; its value is the _OutX modified by the _OffsetX variable.

Default value: 0.0 (Automatic)

_FinalY

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; its value is the _OutY modified by the _RotZ variable.

Default value: 0.0 (Automatic)

_LayerDistance

Type: Numeric parameter.

Function: This variable is automatically calculated by the Layer Mode behavior; it specifies the individual distance used for layer calculations.

Default value: 0.0 (Automatic)

_Layer

Type: Numeric parameter.

Function: This variable is automatically calculated by the Layer Mode behavior; it determines the current layer of the object for the Layer Object.

Default value: 0.0 (Automatic)

_Angle

Type: Numeric parameter.

Function: This variable is automatically calculated by the Distance Mode behavior; it determines the visibility angle of the object based on the _PositionX and _PositionY points.

Default value: 0.0 (Automatic)

_Angular

Type: Numeric parameter.

Function: This variable is automatically calculated by the Mapping Mode behavior; it corrects the _Angle value to ensure it falls within the range 0.0 to 360.0.

Default value: 0.0 (Automatic)

_AngleVisibility

Type: Numeric parameter.

Function: This variable is automatically calculated by the Visible Mode behavior; it specifies the visibility angle of the object.

Default value: 0.0 (Automatic)

_AngularVisibility

Type: Numeric parameter.

Function: This variable is automatically calculated by the Visible Mode behavior; it corrects the _AngleVisibility value to ensure it falls within the range 0.0 to 360.0.

Default value: 0.0 (Automatic)

Alterable Strings
Description
_MappingScreenMode

Type: String parameter.

Function: Responsible for selecting the mode in which the object is displayed on the screen, based on the Mapping Screen Mode behavior.

Default value: DEFAULT

_MappingMode

Type: String parameter.

Function: Responsible for selecting the mode in which the object is mapped, based on the Mapping Mode behavior.

Default value: R1.0.1

_DistanceMode

Type: String parameter.

Function: Responsible for selecting the mode used to calculate distance for the Mapping Mode behavior, based on the Distance Mode behavior.

Default value: DEFAULT

_ScaleMode

Type: String parameter.

Function: Responsible for selecting the mode in which the object is scaled, based on the Scale Mode behavior.

Default value: DEFAULT

_LayerMode

Type: String parameter.

Function: Responsible for selecting which layer the object should reside on, based on the Layer Mode behavior.

Default value: TOPLAYER

_VisibleMode

Type: String parameter.

Function: Responsible for selecting the mode when the object should be visible, based on the Visible Mode behavior.

Default value: DEFAULT

Flags
Description
_Active

Type: Bool parameter.

Function: This variable specifies whether object mapping should be active; when set to True, it continuously updates the positioning on the screen, and when set to False, it does not update the object's position on the screen.

Default value: True

_ShouldVisible

Type: Bool parameter.

Function: This variable is automatically calculated by the Visible Mode behavior; it determines whether the object is visible on the screen based on whether _AngularVisibility is less than 100° or greater than 260°.

Default value: False (Automatic)

_IgnoreShowing

Type: Bool parameter.

Function: This variable specifies whether to disable the automatic visibility control of the object; when set to True, automatic visibility is turned off.

Default value: False

_OutRenderDistance

Type: Bool parameter.

Function: This variable is automatically calculated by the Visible Mode behavior; it indicates whether the _RenderDistance has been exceeded.

Default value: False (Automatic)

_Visible

Type: Bool parameter.

Function: This variable is automatically calculated by the Visible Mode behavior; it determines whether the object should ultimately be visible on the screen.

Default value: False (Automatic)

_QualityScale

Type: Bool parameter.

Function: This variable specifies whether to use anti-aliasing when scaling the object; when set to True, anti-aliasing is applied, and when set to False, scaling is done without anti-aliasing, resulting in a pixelated appearance.

Default value: False
