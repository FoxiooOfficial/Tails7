Tails7 Info Object


1. General Description of Object

Tails7 Mapper Object is an object that stores values such as shader parameters, resolution information and can actulize Tails7 Render Ground


2. Variables

Description

Type: String parameter.

Function: Includes information about the current version of Tails7.

Default value: R1.0.1_13.05.25

_API

Type: String parameter.

Function: Returns information about the current graphics API in use (CPU, DirectDraw, Direct3D8, Direct3D9 or Direct3D11).

Flags
Description
_Update

Type: Bool parameter.

Function: This variable set to True will cause Tails7 Render Ground objects to be deleted and created again to fill the whole screen.

Default value: False

_IsUpdating

Type: Bool parameter.

Function: This flag is set automatically when the create function Tails7 Render Ground is active.

Default value: False (Automatic)



3. Behaviors


3.1. Details: Set API Render in Tails7 Info Object


This behavior will automatically set the appropriate text for _API by miała odpowiedni tekst. Detects for Windows jwhat graphics API the program is using.


List of ways to set _API dependent on DisplayMode:

DisplayMode
Description
0

Returns CPU for _API

1

Returns DirectDraw for _API

4

Returns Direct3D8 for _API

8

Returns Direct3D9 for _API

16

Returns Direct3D11 for _API



3.2. Details: Startup Set Render in Tails7 Info Object


This behavior is responsible for automatically creating Tails7 Render Ground objects on the screen to fill the whole (Can fill the screen by width or height)

This behavior is activated at startup as well as by the _Active flag.


This behavior additionally calculates variables such as: _RenderMinimumSize, _RenderMiddleSizeX, _RenderMiddleSizeY, _ScaleFactorX, _ScaleFactorY, _WidthRatio, _AspectCorrection and _IsUpdating.
