Tails7 Render Ground


1. General Description of Object


This object has a shader that gives the effect of perspective. One of the most important objects because it is used for mapping calculations.


2. Variables

Alterable Values
Description
_RenderOffsetX

Type: Numeric parameter.

Function: This variable is responsible for moving the render in the X axis so that for duplicates it is properly connected.

Default value: 0.0 (Automatic)

_RenderOffsetY

Type: Numeric parameter.

Function: This variable is responsible for moving the render in the Y axis so that for duplicates it is properly connected.

Default value: 0.0 (Automatic)

Flags
Description
_Active

Type: Bool parameter.

Function:This flag determines whether to set and adjust the object to fill it on the screen, this happens when the flag is for False, it is automatically set to True.

Default value: False



3. Behaviors



3.1. Details: Startup in Tails7 Render Ground


This behavior changes object texture and fills the entire screen. This activates when the _Active flag is set to False.

It automatically sets this flag to True after that.

This behavior also disables the visibility of the object if _API has CPU text, DirectDraw or Direct3D8.



3.2. Details: Set Effect Varibles in Tails7 Render Ground


This behavior would always set the appropriate shader parameters from the object Tails7 Info Object variables.
