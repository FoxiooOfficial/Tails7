Player Hitbox


1. General Description of Object

This object can serve as a Hitbox that you give to the player, its position is modified by Tails7 Camera Object. It doesn't have any major functions because it's up to you what you want to do with it.


2. Variables

Alterable Values
Description
_Speed

Type: Numeric parameter.

Function: This variable is used in the behavior of Position. It specifies the speed at which the object moves in the X and Y axes.

Default value: 15.0
