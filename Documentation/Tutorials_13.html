Tutorial 5: Ground Texture Swapping


Here you will learn how to change the texture of the Tails7 Render Ground object.

Tails7 is designed to make it very easy to change the texture.

The first step is to select our Tails7 Render Ground and we open the texture editing window.



It should look like this:



Choose options to create a new animation frame.




Here I paste a sample grass texture. I set the Hot Spot here to be centered because it is needed.



I go to the Event Editor and create such an event that will allow me to change the texture, why 1 and not 2 for _TextureID? The texture index starts at 0 and not 1.
