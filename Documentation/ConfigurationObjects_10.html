Line Objects


1. General Description of Object


These objects are similar to Polygon objects, but it draws lines from point one to point two.


2. Variables

Alterable Values
Description
_PointX_A

Type: Numeric parameter.

Function: This variable sets the beginning of the line position on the screen in the X axis (Automatically calculated by Set Points and Other Varibles behavior).

Default value: 0.0 (Automatic)

_PointY_A

Type: Numeric parameter.

Function: This variable sets the beginning of the line position on the screen in the Y axis (Automatically calculated by Set Points and Other Varibles behavior).

Default value: 0.0 (Automatic)

_PointX_B

Type: Numeric parameter.

Function: This variable sets the end of the line position on the screen in the X axis (Automatically calculated by Set Points and Other Varibles behavior).

Default value: 0.0 (Automatic)

_PointY_B

Type: Numeric parameter.

Function: This variable sets the end of the line position on the screen in the Y axis (Automatically calculated by Set Points and Other Varibles behavior).

Default value: 0.0 (Automatic)

_Thickness

Type: Numeric parameter.

Function: This variable declares the scale of the object in the Y axis.

Default value: 2.0

_TextureSizeX

Type: Numeric parameter.

Function: This variable is for determining what the FACTUAL size of the texture is in the X axis. If the value is too large or too small, the line scale will not be correct.

Default value: 1.0

_LayerOffset

Type: Numeric parameter.

Function: This variable specifies the layer offset: negative values allow the object to render behind objects behind it, and positive values allow it to render above objects in front of it.

Default value: 0.0

_Layer

Type: Numeric parameter.

Function: This variable is automatically calculated by the Set Points and Other Varibles behavior; it determines the current layer of the object for the Layer Object.

Default value: 0 (Automatic)

Flags
Description
_Visibility_A

Type: Bool parameter.

Function:This variable indicates the visibility of the beginning point.

Default value: False (Automatic)

_Visibility_B

Type: Bool parameter.

Function:This variable indicates the visibility of the end point.

Default value: False (Automatic)

_IgnoreShowing

Type: Bool parameter.

Function: This flag sets whether the automatic visibility of the object on the screen should be disabled.

Default value: False

_QualityScale

Type: Bool parameter.

Function: This variable specifies whether to use anti-aliasing when scaling the object; when set to True, anti-aliasing is applied, and when set to False, scaling is done without anti-aliasing, resulting in a pixelated appearance.

Default value: False



3. Behaviors



3.1. Details: Set Points and Other Varibles in Line Objects Objects


This behavior is responsible for setting the _PointX_A, _PointY_A, _Visibility_A, _PointX_B, _PointY_B, _Visibility_B, _Layer variables of the objects specified in it in this behavior.



3.2. Details: Set Position, Rotation & Scale in Line Objects Objects


This behavior is responsible for positioning the object to the assigned point, changes its scale and rotation.



3.3. Details: Set Visibility in Line Objects Objects


This behavior modifies the visibility of the object in the dependency by calculating _Visibility_A * _Visibility_B and the _IgnoreShowing flag.



3.4. Details: Set Texture in Line Objects Objects


This behavior at the start changes the texture of the object to the next one.
