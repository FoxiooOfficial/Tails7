FAQ


Why was this project created?

For those who have used the Mode7ex extension know that it is outdated and can lag. Once upon a time when I was experimenting with shaders, I created a Mode7 shader and came up with the idea why not turn it into an engine? Initially the project was supposed to be released AFTER the release of my game Exadelta, but I changed my mind because you would have to wait a very long time.

Can I distribute a modified version of this project?

Yes, this engine is under MIT License, you can do what you want with it however you must give the author; me, in credits.

Will the engine get support for Android/MAC etc.?

Currently there is a problem with the functioning of the whole thing on Android, I'm trying to get it fixed, support for other platforms will be, however, you will have to wait a very... very long time....

How long will the engine be supported?

The engine will be supported for a long time. I will use it for my projects so if I also find bugs myself I will fix them.

Where to report bugs or ideas?

On the GitHub of the project.

What is Mode7?

Mode7 is a technique to, from a flat two-dimensional texture, distort it to give the effect of perspective
