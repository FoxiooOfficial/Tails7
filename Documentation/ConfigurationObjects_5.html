Tails7 Camera Object


1. General Description of Object


This object is the default object with which Tails7 Render Ground obtains positions, also this object informs about _RotX as the rotation of object.



2. Variables

Alterable Values
Description
_TargetX

Type: Numeric parameter.

Function: This variable is what X position the object should follow, by default it follows the Player Hitbox.

Default value: 0.0 (Automatic)

_TargetY

Type: Numeric parameter.

Function: This variable is what Y position the object should follow, by default it follows the Player Hitbox.

Default value: 0.0 (Automatic)

_OffsetX

Type: Numeric parameter.

Function: This value is the object's X axis offset

Default value: 0.0

_OffsetY

Type: Numeric parameter.

Function: This value is the object's Y axis offset

Default value: 0.0

_Radius

Type: Numeric parameter.

Function: This variable is the distance between the target position.

Default value: 0.0

Alterable Strings
Description
_PositionMode

Type: String parameter.

Function: Responsible for how the camera should behave, i.e. how to set the position.

Default value: FIRST_PERSON_LERP

3. Behaviors



3.1. Details: Position in Tails7 Camera Object


This behavior has built in events that set Player Hitbox positions and changes varibles from Tails7 Info Object In it we also set _TargetX and _TargetY from position Player Hitbox.



3.2. Details: Position Mode in Tails7 Camera Object

This behavior determines how the object should behave, i.e. setting its position as well as shader variables, it depends on _PositionMode

List of available modes:

Mode
Description
DEFAULT

Default mode which does not use any modifications, camera moves backwards relative to _PosZ.

FIRST_PERSON

A mode that is geared to the first person, moves the camera forward so that it stays in one place at _PosZ.

FIRST_PERSON_LERP

A mode similar to FIRST_PERSON however uses linear interpolation and the transition between _Distortion is smooth.
