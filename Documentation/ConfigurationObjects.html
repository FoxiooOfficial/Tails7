Configuration of Objects

This page provides essential information about key embedded objects and their basic functions.

Icon
Description
Link

Player Hitbox

Can be used as collision detection object for checking interactions with other game elements.

More Info

Tails7 Info Object

Initializes and creates the Tails7 Render Ground object. Contains basic configuration settings.

More Info

Tails7 Render Ground

Main Mode7-style shader object creating perspective projection. Most calculations are based on this object.

More Info

Tails7 Mapper Object

Responsible for all calculations and mapping of 3D.Objects (i.e. Sprite Objects) on the screen.

More Info

Tails7 Camera Object

Sets the coordinates of the X/Y position and reports the rotation of the X axis.

More Info

Tails7 Center Pointer

Automatically selects default Render Ground and positions itself at screen center.

More Info

Tails7 Background Gradient

Objects that overlay a gradient on the screen.
External: Applies a gradient from the shader horizon to the top.
Internal: Applies a gradient from the shader horizon to the bottom.

More Info

Sprite Objects

Basic elements that get perspective-mapped to screen coordinates.

More Info

Polygon Objects

Shader-based 3D wall illusion using quad geometry.
Lonely: Self-contained with predefined points
Requiring: Needs manual point declaration

More Info

Line Objects

Draws simple lines between two specified screen points.
